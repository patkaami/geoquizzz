About GeoQuizzz

GeoQuizzz is a simple quiz game created as a school project. The game features four categories: Capitals, Flags, Trivia, and Mix. In each category, you must guess the correct answer based on the question presented. Before starting, you are asked to enter a nickname to track your progress. You can view your past quizzes in the History section, where you can select previously played quizzes and review them in Review Mode.
Note that the history only shows quizzes played in your current browser. The app is built with JavaScript, using the REST Countries API, jQuery, and interactive alerts powered by SweetAlert2.
